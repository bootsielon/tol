
Para compilar con MSVC se recomienda instalar:

 + MS Visual Studio 2008 Express (VS2008ExpressENUX1397868.iso)
   https://go.microsoft.com/fwlink/?LinkId=104679

 + MS Windows SDK para Windows 7:
   - GRMSDK_EN_DVD.iso    -- x86 environment.
   - GRMSDKX_EN_DVD.iso   -- x64 environment.
   - GRMSDKIAI_EN_DVD.iso -- Itanium environment.
   https://www.microsoft.com/en-us/download/details.aspx?id=8442
